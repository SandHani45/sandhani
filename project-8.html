IBE-Optics

Client : IBE-Optics
Completion : March 2018 - Present
Skils : HTML5,CSS3,BOOTSTRAP,JAVASCRIPT,REACT, NODE JS, EXPRESS JS

Smartfinder replaces the traditional viewfinder with an easier to use and a way more accurate and adaptable set of features. The user first needs to select a camera format and lens grouping, Smartfinder simulates the view of the lens you might use on your shooting a scene. The user can either compare the lens selection in a wireframe view or the specific focal length in zoom view.
